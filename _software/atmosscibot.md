This twitter bot generates word clouds of new open access publications in atmospheric sciences. The bot parses RSS feeds of scientific journals, downloads new publications in HTML/XML and then builds a word cloud image of the extracted text.

The source code is publicly available on [GitHub](https://github.com/dennissergeev/atmosscibot) under MIT license.
