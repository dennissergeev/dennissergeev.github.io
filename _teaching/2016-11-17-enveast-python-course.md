This was a 3-day workshop funded by [EnvEast DTP](http://www.enveast.ac.uk/) and [Cefas](http://www.cefas.co.uk/) will give a hands-on introduction to [Python](http://www.python.org/) programming language. It focused on the basics of the analysis and visualisation of the data commonly used in environmental sciences, starting from the basics.

Topics that we covered include
* Python basics and programming environment
* Core scientific libraries: numpy, scipy, pandas
* Simple plots in matplotlib
* Plotting data on maps
* Working with netCDF data

The course content is freely available as
* static notebooks on [nbviewer](http://nbviewer.jupyter.org/github/ueapy/enveast_python_course_materials/blob/master/index.ipynb)
* or interactive notebooks in the cloud [![Binder](http://mybinder.org/badge.svg)](http://mybinder.org:/repo/ueapy/enveast_python_course_materials)
